# 2. 빅데이터-수학,통계,도구

### 빅데이터

- 엄청나게 많은 양의 데이터
- 대량으로 축적된 데이터



### 머신러닝

- 수단
- 빅데이터를 이용해 데이터를 분석하는 기법 중 하나
- 알고리즘 선택/분석 단계
- 어떻게 목표에 원하는 수치에 도달할 것인가
- 수학적 방법론

1. 목적 정의
2. 목적에 맞는 데이터가 있어야함
3. 어떤 머신러닝 기법을 쓸지 선택



- 데이터를 저장할 공간(용량, 저장장치)가 충분하지 않았음
- 저장장치 기술의 발전으로 최근 핫해짐



### 필요 스킬

1. 통계
   - 표준편차
2. 수학
   - 선형대수(행렬 확장)
   - 미분
3. 도구(Tool)
   - 프로그래밍 언어 기술

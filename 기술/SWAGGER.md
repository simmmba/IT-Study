# SWAGGER

- API 개발 툴
- Spring과 높은 호환성(Flask도 사용 가능)
- 삼성전자, 네이버, 카카오, NHN 등 대부분의 회사에서 사용
- REST API 쭉 볼 수 있고
- POST MAN 기능 제공(input output 결과 볼 수 O)
- 백엔드개발자와 프론트엔드 개발자의 소통 도구
- 개발자는 Interface로 소통
- API 정의, 공유, Test, 문서화에 적합한 도구

<br />

### 면접에 적용

- 개발, 협업 경험에 얘기 가능.

- Backend 개발자로, Front와 협업을 위해 SWAGGER를 사용했습니다.
